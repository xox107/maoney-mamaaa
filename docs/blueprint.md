# **App Name**: MONEY MAMAA

## Core Features:

- Add Income: Add income entries with amount, category, note, and a 'Pending Payment' checkbox. Date is automatically timestamped.
- Add Expense: Add expense entries with date, amount, category, and note.
- Pending/Confirmed Income: Display income in two sections: Pending and Confirmed. Allow users to toggle the 'Pending Payment' checkbox.
- Dashboard Summary: Display total income, total expenses, and remaining balance.
- Category Breakdown: Provide a breakdown of expenses and income by category.
- Monthly Summary & Chart: Generate a monthly summary and a simple bar or pie chart for visualizing financial data.

## Style Guidelines:

- Primary color: Deep Blue (#283593) to convey trust and security.
- Background color: Light Blue-Gray (#E8EAF6), almost white, to create a calm and clean backdrop.
- Accent color: Subtle Orange (#EF6C00) to highlight key interactive elements.
- Body and headline font: 'PT Sans', a humanist sans-serif combining a modern look with a touch of warmth; appropriate for both headlines and body text
- Simple, outlined icons for categories and actions, aligning with a clean UI.
- Clean, card-based layout for transactions and summary information. Mimic Google Keep's checkbox style.
- No animations.
